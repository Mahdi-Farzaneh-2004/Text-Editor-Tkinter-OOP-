# 📝 Professional Text Editor (Tkinter & OOP)

یک ویرایشگر متن (Text Editor) دوزبانه، شیک و کامل که با استفاده از **Python** و کتابخانه **Tkinter** بر پایه اصول شیءگرایی (OOP) طراحی و پیاده‌سازی شده است.

---

## ✨ ویژگی‌های پروژه (Features)

* **🌐 کاملاً دوزبانه:** پشتیبانی کامل از تغییر زبان رابط کاربری بین **فارسی (Fa)** و **انگلیسی (En)** تنها با یک کلیک.
* **🎨 شخصی‌سازی ظاهر:** امکان تغییر رنگ متن (Foreground) و رنگ پس‌زمینه (Background) به صورت داینامیک.
* **🔤 مدیریت فونت پیشرفته:** قابلیت تغییر نوع فونت، سایز قلم و اعمال استایل‌های **Bold** و *Italic*.
* **🔢 شماره‌گذاری خودکار خطوط:** دارای سیستم هوشمند هماهنگ با کلید `Return` برای اضافه کردن فرمت‌های مختلف شماره‌گذاری (`1.`، `1:`، `1=`).
* **📊 نوار وضعیت (Status Bar):** نمایش دقیق موقعیت نشانگر موس بر اساس شماره خط (Line) و ایندکس کاراکتر.
* **💾 مدیریت فایل کامل:** قابلیت ساخت فایل جدید (`New`) با نام‌گذاری خودکار، باز کردن فایل‌های متنی (`Open`)، ذخیره سریع (`Save`) و ذخیره با نام دلخواه (`Save as`).
* **🧩 ساختار استاندارد:** کدنویسی تمیز، ماژولار و کاملاً شیءگرایی (Object-Oriented).

## 🇬🇧 English Description

A feature-rich, bilingual Text Editor built using **Python** and **Tkinter**, designed entirely on Object-Oriented Programming (OOP) principles.

### ✨ Features
* **🌐 Bilingual Support:** Toggle the entire user interface language between **Persian (Fa)** and **English (En)** instantly.
* **🎨 UI Customization:** Change text color (Foreground) and editor background color dynamically.
* **🔤 Advanced Font Management:** Easily modify font families, font sizes, and toggle **Bold** or *Italic* styles.
* **🔢 Auto-Numbering System:** Smart paragraph/bullet numbering (`1.`, `1:`, `1=`) automatically triggered on `Return` (Enter) key.
* **📊 Live Status Bar:** Displays the exact real-time cursor position by Line and Character Index.
* **💾 File Management:** Full support for creating new files (`New`) with auto-incrementing names, opening text files (`Open`), quick save (`Save`), and (`Save as`).
* **🧩 Clean Architecture:** Well-structured, modular, and fully object-oriented codebase.
